# JSON 格式化工具

一个简洁的在线 JSON 格式化工具，单文件实现，无需依赖。

## 功能

- **格式化** - 将 JSON 美化输出，支持 2/4 空格缩进
- **压缩** - 将 JSON 压缩为单行
- **复制** - 一键复制处理结果
- **过滤 null** - 可选忽略 JSON 中的 null 值
- **粘贴自动格式化** - 粘贴内容后自动格式化

## 使用

直接在浏览器中打开 `index.html` 即可使用。

```bash
# Linux
xdg-open index.html

# macOS
open index.html

# Windows
start index.html
```

## 特点

- 单文件，无外部依赖
- 响应式布局，适配移动端
- 自适应高度，充分利用屏幕空间
- 浅色主题，清爽简洁
